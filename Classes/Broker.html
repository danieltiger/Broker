setRootKeyPath:forEntity:

Set the root key path for a given entity previously registered

- (void)setRootKeyPath:(NSString *)rootKeyPath forEntity:(NSString *)entity

Parameters

rootKeyPath

The root key path of the entity resource.

entity

The name of the entity, previously registered with Broker

Discussion

Set the root key path for a given entity previously registered

The root key path is useful when the returned resources are nested. For a resource named “User,” the JSON might look like the following.

{ 
    'response' : { 
        'user' : {
                  <DATA>
                 }
    }
}

In this case, the rootKeyPath would be @“response.user”.
